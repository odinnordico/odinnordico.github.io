# Welcome to the odinnordico.github.io Wiki!

**odinnordico.github.io** is a powerful, flexible resume generator built with Go. It allows you to create professional PDF resumes and responsive static websites from simple YAML data files. Designed for developers, it supports version control, multi-language resumes, and extensive customization.
